# 🚀 Bootcamp Java & IA — A Jornada do Builder

Bem-vindo! Este é o repositório oficial do bootcamp.
Aqui você encontra tudo que precisa para acompanhar as 12 semanas.

## 🎯 O que você vai construir
Um sistema funcional, em squad, do zero ao Demo Day.
Não é sobre virar sênior em 12 semanas — é sobre saber **começar do jeito certo**
e ter autonomia para continuar evoluindo depois.

## 🗺 Como navegar
- [`roadmap.md`](./roadmap.md) — visão macro dos 5 níveis
- [`schedule.md`](./schedule.md) — agenda das 12 aulas
- [`weeks/`](./weeks/) — conteúdo de cada semana
- [`templates/`](./templates/) — modelos para o projeto final
- [`demo-day/`](./demo-day/) — como funciona a apresentação final

## 👥 Sua organização (squad)
Em squad de até 3 pessoas, vocês criam **sua própria GitHub Organization**
com 2 repositórios:
- `practice` — exercícios da semana
- `projeto-final` — o sistema que evolui semana a semana

## 🧙 Método: Wizard + 🧠 Second Brain
- **Wizard** — aprender fazendo, errar, repetir
- **Second Brain** — IA acelera, mas você decide
- **Squad** — "ninguém fica para trás"
- **MVP > tudo** — entrega que roda > código que impressiona

## 💬 Canal de dúvidas
Grupo de WhatsApp livre e horizontal: qualquer um responde, qualquer um ajuda.

## ✅ Como aproveitar ao máximo
1. Antes da aula: lê o README da semana em `weeks/week-XX-*/`
2. Durante a aula: pratica em squad
3. Depois da aula: termina a lição de casa e commita na sua org
4. Não terminou? Sobe como WIP — a revisão coletiva é no Demo Day

Boa jornada, Builder. 🚀
