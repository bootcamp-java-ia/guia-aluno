# 📋 GitHub Project — Template

Sugestão para organizar as tarefas do squad ao longo das semanas.

## Colunas
- **Backlog** — ideias e itens que ainda não entraram em uma semana
- **A Fazer (semana atual)** — o que será feito esta semana
- **Em Andamento** — alguém do squad já está mexendo
- **Em Revisão** — PR aberto, esperando outra pessoa do squad revisar
- **Concluído** — mergeado e funcionando

## Convenção de cards
Cada card representa **uma tarefa entregável**:
- `[Sem 05] Criar classe Produto`
- `[Sem 09] Salvar lista em arquivo CSV`
- `[Sem 11] Endpoint GET /produtos`

## Boas práticas
- 1 card = 1 PR
- Mover o card no quadro **antes** de começar a tarefa
- Cards travados há mais de 2 dias → leve ao grupo

## Como criar
1. Em `github.com/<sua-org>` → **Projects** → **New project**
2. Escolher template **Board**
3. Renomear colunas conforme acima
4. Conectar com os repos `practice` e `projeto-final`
