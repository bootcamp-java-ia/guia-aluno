# 📚 Templates – Bootcamp Java & IA

Modelos oficiais que os squads devem usar durante o bootcamp.

## 🎯 Objetivo
Facilitar o início dos projetos, reduzindo atrito para o squad focar no que importa:
**aprender, praticar e entregar.**

## 📁 Conteúdo

- 📄 [Lean Canvas](./lean-canvas.md) — definir problema, solução, público, valor.
  **Obrigatório no projeto final.**
- 📄 [README do Projeto](./README-projeto.md) — modelo de documentação do squad.
- 📄 [Estrutura de Apresentação](./apresentacao.md) — guia para o Demo Day
  (apresentação ao vivo ou vídeo).
- 📄 [GitHub Project Template](./github-project.md) — organização das tarefas.

## 🧠 Importante
Templates **não são burocracia**. Eles ajudam a:
- pensar melhor o problema
- organizar o trabalho em equipe
- comunicar a solução

## ✅ Regra de Ouro
> **Simples, organizado e funcional** sempre vence **complexo e incompleto.**
